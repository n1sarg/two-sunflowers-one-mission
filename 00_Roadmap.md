# 📖 Storybook Generator — Complete Course Roadmap

> **"We're not just learning AI — we're building, deploying, and shipping a real product."**

This is an accelerated daily-sprint course. No fluff, no waiting for weekends.
You go as fast as you can, one exercise per day minimum, and you don't skip anything.
By the end, you'll have built an **AI-powered multi-agent storybook generator**
deployed to **Azure** with a full **CI/CD pipeline** across dev/UAT/prod environments.

That's not a tutorial project. That's a portfolio-grade production system.

---

## 🏗️ The Five Phases

```
Phase 1: FOUNDATIONS (Days 1–5)          → Talk to LLMs, learn the tools
Phase 2: BUILD THE ENGINE (Days 6–9)     → Structured output, story pipeline, web UI
Phase 3: AGENTIC AI (Days 10–14)         → Tool use, agents, multi-agent orchestration
Phase 4: DEVOPS & CONTAINERS (Days 15–18)→ Docker, environments, infrastructure
Phase 5: AZURE & CI/CD (Days 19–22)      → Cloud deployment, pipelines, production
```

## 📅 Daily Sprint Schedule

### Phase 1: FOUNDATIONS (Days 1–5)
*Goal: Be comfortable talking to LLMs and using the core tools.*

| Day | Exercise | What You Learn | Commit |
|-----|----------|---------------|--------|
| 1 | `01_hello_llm.py` | API calls, system prompts, Rich | 🎉 First AI conversation! |
| 2 | `02_playlist_namer.py` + `03_token_counter.py` | Temperature, costs, tokens | 💰 I know what AI costs |
| 3 | `04_streaming_poetry.py` | Streaming, real-time output | ✨ Text appears like magic |
| 4 | `05_langchain_basics.py` | LangChain, chains, templates | 🔗 First chain works |
| 5 | `06_pdf_reader.py` + `07_text_file_buddy.py` | Document processing | 📄 Chat with my docs! |

### Phase 2: BUILD THE ENGINE (Days 6–9)
